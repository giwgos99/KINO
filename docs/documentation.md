# Docs

Καλό θα ήταν πιστεύω ο κάθε ένας να φτιάξει το δικό του κομμάτι του documentation, δηλαδή το τι κάνει στην εργασία του κτλ. π.χ. ένα για GUI, ένα για Logic, ένα για database. Ο κάθε ένας να φτιάξει το δικό του.
Επίσης θα βοηθήσει και γιατί ο κάθε ένας πρέπει να παραδώσει και δικό του ξεχωριστό word για την εργασία.

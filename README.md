# KINO

Καλησπέρα σε όλους! 
Το συγκεκριμένο Repository έχει χρησιμοποιηθεί για την υποβολή ενός ομαδικού project στο Ελληνικό Ανοικτό Πανεπιστήμιο (ΕΑΠ). 
Το θέμα του project είναι να φτιάξουμε μια εφαρμογή ΚΙΝΟ, η οποία να υποστηρίζει και κάποιες συγκεκριμένες λειτουργίες που έχουν καθοριστεί απο τον Καθηγητή μας.

Υπάρχει επίσης βίντεο χρήσης του project, στο url: 
https://www.youtube.com/watch?v=a8CNqwJvBDM

Ευχαριστούμε!
